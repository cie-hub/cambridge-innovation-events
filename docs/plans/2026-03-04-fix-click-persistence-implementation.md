**Goal:** Fix two bugs that cause click analytics data to be lost on every scrape: the `clicks` counter on events getting overwritten, and the `clicks` collection storing `ObjectId` references that break when events are re-upserted.

**Architecture:** Two targeted fixes — (1) change the scrape upsert from `$set` (full replace) to `$set` + `$setOnInsert` so the `clicks` field is preserved on updates, and (2) change the track endpoint to store `hash` (stable identifier) instead of `_id` (unstable ObjectId) in the `clicks` collection.

**Tech Stack:** Node.js ESM, MongoDB, Vitest

**Step 3: Update track.js**

```js
import { ObjectId } from 'mongodb'
import { getDb } from './_lib/db.js'

export default async function handler(req, res) {
  if (req.method !== 'POST') return res.status(405).end()

  const { eventId } = req.body || {}
  if (!eventId || !ObjectId.isValid(eventId)) return res.status(400).end()

  try {
    const db = await getDb()
    const oid = new ObjectId(eventId)

    const event = await db.collection('events').findOne({ _id: oid }, { projection: { hash: 1 } })
    if (!event) return res.status(404).end()

    await Promise.all([
      db.collection('events').updateOne({ _id: oid }, { $inc: { clicks: 1 } }),
      db.collection('clicks').insertOne({ hash: event.hash, timestamp: new Date() }),
    ])

    return res.status(204).end()
  } catch {
    return res.status(500).end()
  }
}
```

### Task 3: Migrate existing click records (if any)

Write a one-off script that finds any existing `clicks` documents with `eventId` (old format) and adds the `hash` field by looking up the event. Run once, then delete.

**Files:**
- Create: `scripts/migrate-clicks.js` (temporary, delete after running)

**Step 1: Write the migration script**

```js
import { MongoClient, ObjectId } from 'mongodb'
import 'dotenv/config'

const client = new MongoClient(process.env.MONGODB_URI)
await client.connect()
const db = client.db('cambridge-events')

const clicks = db.collection('clicks')
const events = db.collection('events')

const oldClicks = await clicks.find({ eventId: { $exists: true }, hash: { $exists: false } }).toArray()
console.log(`Found ${oldClicks.length} clicks to migrate`)

let migrated = 0
for (const click of oldClicks) {
  const event = await events.findOne({ _id: click.eventId })
  if (event) {
    await clicks.updateOne({ _id: click._id }, { $set: { hash: event.hash }, $unset: { eventId: '' } })
    migrated++
  } else {
    await clicks.deleteOne({ _id: click._id })
  }
}

console.log(`Migrated: ${migrated}, orphaned (deleted): ${oldClicks.length - migrated}`)
await client.close()
```
